# ❓ Installation

Welcome to the installation guide for Dadi Exotic Paints.

<Aside type="caution">
You must have **ESX version 1.9** or newer (or a compatible framework like QBCore/Qbox) for the script to work. Remember to clear your cache after installation.
</Aside>

## Dependencies
This script is **free of dependencies**.

## How to Install the Script

<Steps>
1.  **Download the script** from your FiveM keymaster. You will receive a .zip file named `dadi-paints.pack.zip`.
2.  **Open** the compressed file.
3.  **Create a folder** named `[dadi-market]` (if you don't have it yet) in your server's resources directory.
4.  **Copy** the `dadi-paints.pack.zip` file into the newly created folder.
5.  **Extract here**. Ensure that the folder `dadi-paints` is generated.
6.  **Clear the server cache** before starting the server.
7.  **Start your FiveM server** and ensure the script is loaded in your `server.cfg`.
</Steps>

---

<Aside type="note">
Following these steps will complete the installation of the Dadi Exotic Paints script on your FiveM server. Now, the script will be ready to use for your players.
</Aside>
