## How to Install the Script

<Steps>
1.  **Download** the script from your FiveM keymaster.
2.  **Open** the compressed file and extract the contents.
3.  **Place** the folder in your `[resources]` directory.
4.  **Rename** it to `script_name` (if specific naming is required).
5.  **Edit** your `server.cfg` to ensure the script is loaded.
</Steps>
